Poslaju Tracking 
=========================

Features
--------
* Tracking poslaju package

Installation
------------
Require this package with composer:
```
composer require achmadfatoni/poslajuphp
```

Usage
------
```
$poslaju = new Achmadfatoni\Poslaju\Poslaju();  
$track = $poslaju->track('trackingNumberHere')
```
